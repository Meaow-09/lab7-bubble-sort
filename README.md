# Bubble Sort Lab

A simple Python implementation of the Bubble Sort algorithm, with optional terminal visualization.

## Project Files

- `main.py` — bubble sort implementation and visualization helpers
- `test_main.py` — basic pytest tests
- `REPORT.md` — lab report / notes
- `JOURNAL.md` — interaction and change log

## Requirements

- Python 3.10+ (3.12 also works)
- `pytest` (for running tests)

## Run the Program

```bash
python main.py
